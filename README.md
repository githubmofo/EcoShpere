# EcoSphere – ESG Management Platform

An enterprise-grade ESG (Environmental, Social, Governance) management dashboard designed to track sustainability metrics, foster corporate social responsibility, and ensure governance compliance. Built with a modern tech stack featuring **Next.js**, **Tailwind CSS**, **shadcn/ui**, **Recharts** on the frontend, and an **Express + Prisma + SQLite** backend.

## 📂 Project Structure

```text
ecosphere-esg-platform/
├─ frontend/    # Next.js App Router (Tailwind + shadcn/ui + Recharts)
├─ backend/     # Express API + Prisma ORM + SQLite database
└─ README.md
```

## 🚀 Getting Started

Follow these step-by-step instructions to install, configure, and run the EcoSphere platform locally. You will need to run two separate terminal windows—one for the backend and one for the frontend.

### 1. Setting up the Backend (Database & API)

The backend uses Prisma with a local SQLite database for easy setup.

Open your first terminal and run the following commands:

```bash
# Navigate to the backend directory
cd backend

# Install all backend dependencies
npm install

# Generate the Prisma Client
npx prisma generate

# Create the SQLite database and apply the schema
npx prisma db push

# Seed the database with default Admin and Employee users
npx tsx create-users.ts

# Start the backend development server
npm run dev
```

> **Note:** The backend server will run on `http://localhost:5000` by default.

### 2. Setting up the Frontend (UI)

Open a **second** terminal window and run the following commands:

```bash
# Navigate to the frontend directory
cd frontend

# Install all frontend dependencies
npm install

# Start the Next.js development server
npm run dev
```

> **Note:** The frontend server will run on `http://localhost:3000` by default.

## 📖 Platform Usage Guide

### ➕ Adding Data
Throughout the platform, you can manually input or update ESG metrics. Look for the action buttons (often located at the top right of module pages or inside cards):
- **Environmental:** Add new Carbon Emission Transactions or log new Environmental Goals.
- **Social:** Log new Corporate Social Responsibility (CSR) activities.
- **Settings:** Configure ESG scoring weights, add departments, or update notification preferences.

### 📥 Downloading Reports
The platform includes a robust **Custom Report Builder** designed for stakeholders and auditors.
1. Navigate to the **Reports** page from the main sidebar.
2. Select the specific modules you want to include (Environmental, Social, Governance, Gamification).
3. Choose your preferred format:
   - **Download PDF:** Generates a beautifully formatted document containing charts and structured data.
   - **Download Excel:** Exports the raw tabular data as an `.xlsx` file for spreadsheet analysis.

### 🎮 Gamification & Challenges
Employees can navigate to the **Gamification** tab to:
- View their current XP and point balances.
- Participate in active sustainability challenges.
- Redeem earned points for rewards in the rewards store.

## 🛠️ Tech Stack

### Frontend
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + custom glassmorphic aesthetics
- **Components:** shadcn/ui + Radix Primitives
- **Charts:** Recharts
- **Icons:** Lucide React

### Backend
- **Framework:** Node.js + Express
- **Database:** SQLite (local `dev.db`)
- **ORM:** Prisma
- **Security:** bcrypt/crypto hashing for authentication
